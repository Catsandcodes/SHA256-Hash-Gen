# File Hash Gen

This Python script allows you to calculate the SHA-256 hash of a selected file. It's a useful tool for verifying file integrity and authenticity.

## How to Use

1. Run the script.
2. A file dialog will appear. Select the file you want to hash.
3. The script will print the SHA-256 hash of the selected file.

## Requirements

- Python 3.x
- `tkinter` (usually included with Python standard library)
- `hashlib` (part of Python standard library)
